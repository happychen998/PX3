导航

Bootstrap中的导航组件分享了常用的标记和样式，从基本的.nav类到激活和禁用的状态。交换修饰类，可以在各种样式之间切换。

简易导航

纵向

  • 第一个链接
  • 第二个链接
  • 第三个链接
  • 无效链接
    //- source code
    .row.bd-example
        ul.nav
            li.nav-item: a.nav-link(href="#") 第一个链接
            li.nav-item: a.nav-link(href="#") 第二个链接
            li.nav-item: a.nav-link(href="#") 第三个链接
            li.nav-item: a.nav-link.disabled(href="#") 无效链接

横向

第一个链接  第二个链接  第三个链接  无效链接
    //- source code
    .row.bd-example
        nav.nav
            a.nav-link.active(href="#") 第一个链接&nbsp;&nbsp;
            a.nav-link(href="#") 第二个链接&nbsp;&nbsp;
            a.nav-link(href="#") 第三个链接&nbsp;&nbsp;
            a.nav-link.disabled(href="#") 无效链接

选项卡

将上面的基本导航放在一起，添加一个.nav-tabs类以生成一个可轻触的界面。用 轻触JavaScript 插件创造可轻触的区域。

在其中增加了一个下拉菜单，属性中有关role,aria-haspopup,aria-expanded等是为了无障碍访问定义的属性

  • 第一个链接
  • 第二个链接
  • 第三个链接
  • 下拉菜单
    第一个下拉菜单第二个下拉菜单
    第三个下拉菜单
  • 无效链接
    //- source code
    .row.bd-example
        ul.nav.nav-tabs
            li.nav-item
                a.nav-link(href="#") 第一个链接
            li.nav-item
                a.nav-link.active(href="#") 第二个链接
            li.nav-item
                a.nav-link(href="#") 第三个链接
            li.nav-item.dropdown
                a.nav-link.dropdown-toggle(href="#",data-toggle="dropdown",role="button",aria-haspopup="true",aria-expanded="false") 下拉菜单
                .dropdown-menu
                    a.dropdown-item(href="#") 第一个下拉菜单
                    a.dropdown-item(href="#") 第二个下拉菜单
                    .dropdown-divier
                    a.dropdown-item(href="#") 第三个下拉菜单
            li.nav-item: a.nav-link.disabled(href="#") 无效链接
